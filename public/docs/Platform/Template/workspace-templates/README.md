# 워크스페이스 템플릿 파일

이 디렉토리의 파일들을 복사하여 통합 워크스페이스 디렉토리에 사용하세요.

## 사용 방법

1. `c:\project\spark-messaging-workspace` 디렉토리 생성
2. 이 디렉토리의 파일들을 복사:
    - `spark-messaging.code-workspace.template` → `spark-messaging.code-workspace`
    - `.cursorrules.template` → `.cursorrules`
3. `spark-messaging.code-workspace` 파일에서 경로 수정 (필요시)
4. Cursor에서 워크스페이스 파일 열기

자세한 내용은 `WORKSPACE_SETUP.md` 참고
